双指针与滑动窗口

  1. 基于数组结合滑动窗口，形成的极值问题；
  2. 基于数组，结合贪心的双指针问题。

双指针

11. 盛水最多的容器

每次以双指针为边界，得到最大容量。

双指针正确性：假设当前x<y, res = min(x, y) * t = x * t

  • 假设移动右边，导致长度会减小，高度上限为x；后序只能变小。
  • 假设移动左边：导致长度会减小，高度上限为y；后序可以变大。获得更高的可能性更大，因此这是移动的方向。

接雨水

42. 接雨水I

407. 接雨水II

可以接住水：

  • 非最外层方块
  • 最低盆地

木桶原理：维护最小堆，最外层优先入堆。基于最小堆 + BFS，可以实现每次都是从最小的位置开始搜索。逐步实现低处的雨水累加。

xx. 查找子序列

给定target数值，返回可能的子序列，使得其和为target。

220. 存在重复元素III

判断是否存在两元素下标和差均满足条件。
基于桶排序 + 窗口，维护一个滑动窗口，对于元素x，其影响的区间为[x - t, x + t]。于是我们可以设定桶的大小为t + 1。

  1. 如果两个元素同属一个桶，那么这两个元素必然符合条件;
  2. 如果两个元素属于相邻桶，那么需要校验两元素的差值不超过t。
