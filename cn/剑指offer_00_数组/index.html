一. 查找

3. 数组中重复的数

元素范围与下标有关系

3.1 查找数组N个数

  1. 数值范围：0～N-1；
  2. 将元素放置对应下标，重复即找到；
  3. 每个元素：被动移走 + 主动归位，所以一共两次；
  4. 复杂度为O(2N)

3.2 不修改数组查找 N+1个数（1～N）

  1. 如果可修改数组，按照上述办法也可，nums[i]!=i+1
  2. 不修改数组：O(N)的空间复杂度，统计次数；
  3. 时间换空间思路，如下

时间换空间-只能找出1个

  1. 固定变量统计次数O(1)；
  2. 假设重复数是x，在1～N的二分范围内{a,mid}，O(logN)，统计符合该范围{a,mid}元素的次数O(N)；逐步缩小重复数的区间。因此复杂度为O(NlogN)
    1. 左侧区间 < 次数：
      1. 左侧区间超了，左边肯定重复
    2. 左侧区间 == 次数：
      1. 左侧不能确定是否存在重复；
      2. 但右侧N-M区间里，有N-M+1个数，那么确定右边肯定重复，则left右移至mid+1。
    3. 左侧区间 > 次数：
      1. 左侧次数不够，右边肯定重复，，则left右移至mid+1；

4. 二维数组的查找

  1. 矩阵：左到右递增；上到下递增。使用二分方法，递归的选择每一步走向；
  2. 左下角，或者右上角复杂度为O(M+N)；
  3. 由于矩阵上下均有序，故可以使用二分O(logM+logN)；

11. 旋转数组最小值

  1. 虽然是旋转数组，但因为仍存在有序部分，故依然用二分方法；
  2. 右侧为最小值的递增区间，如果mid-1会错过；
  3. 左侧为大值区域，故mid+1；
  4. 如果当前相等，右侧左移即可；
  5. 复杂度最大为O(N)，最小为O(logN)。

21. 奇数置于偶数前面

  1. 左找偶；右找奇；置换即可；
  2. 复杂度为O(N)。

29. 顺时针打印

39. 数组超过一半的数字

  1. 投票法
    1. 初始化res=nums[0]和票数cur_sum=1；
    2. 相同则票数+1，不同则票数-1，如果0票则当前重新初始化；

40. 数组中最小的k个数

  1. 时间复杂度：期望为O(n)，参考《算法导论》第9章第2小节，最坏情况下的时间复杂度为O(n^2)；
  2. 情况最差时，每次的划分点都是极值，需要划分 n−1 次，最大时间复杂度为O(n^2)；
  3. 空间复杂度：期望为O(logn)，递归调用的期望深度为O(logn)，每层需要的空间为O(1)，只有常数个变量。
  4. 最坏情况下的空间复杂度为O(n)。最坏情况下需要划分n次，即 randomized_selected 函数递归调用最深 n−1 层，所以一共需要 O(n) 的空间复杂度。

40. 最小的第K个数

基于上述方式，得到最小的K个数，进而求极值。总的复杂度为：O(N) + O(k)

41 数据流中的中位数

  1. 最大堆存放较小的一半数（small_heap）；最小堆存放较大的一半数（big_heap）；
  2. 取small_heap的top或者(small_heap.top + big_heap.top)/2

53. 有序数组的元素个数

  1. 递增数组，二分查找；基于lower_bound和upper_bound得到下标位置差。
  2. lower_bound(nums.begin(),nums.end(),target)返回第一个大于或等于target的指针；
  3. upper_bound(nums.begin(),nums.end(),target)返回第一个大于target的指针。

53. 0～n-1缺失的数字

范围是0～n的数组，实际有n个数，寻找缺失值？

  1. 数学方法：(n+1) * n / 2 - sum(arr)

56. 数组出现1次的两个数【其他两次】

  1. 所有元素异或得到ret
  2. 得到第一个为1的bit；
  3. 分组异或，相同的数最终为0；分别得到bit位1和0的两个。

56. 数组出现1次的数【其他三次】

  1. 每个数，按照二进制位累加；
  2. 每个位对3取余数（非0即1）；
  3. 将余数按照二进制转10进制即可。
  4. 可以扩展为，数组中数字出现n次，有个只出现1次

57.和为s的两个数

  1. 递增数组，两头逼近

二. 动态规划与递归

42. 连续子数组最大和

  1. 起始为num[0]；
  2. 当前和为cur_sum = max(cur_sum +nums[i], nums[i]
  3. 最大和为res = max(cur, res)

45. 把数组排成最小的数

  1. 实际考察的是，字符串的排序；一般排序复杂度为NlogN，判断条件是x<y；
  2. 自定义排序：sort(strs.begin(), strs.end(), [](string &x, string &y) {return x + y < y + x;});

46. 数字翻译为字符串的种类【动态规划】

  1. 归纳翻译规则：
    1. 把当前i位当作一位数，那么当前次数就是取f(i-1)；
    2. 如果把当前第1位和第i-1位组成一位，且在10到25之间，那么当前的次数就是取f(i-2)；
  2. 因此：$f(i) = f(i-1) +f(i-2) \{10<= x(i,i-1) <=25\}$

51. 数组的逆序对

利用归并排序中，合并过程中，left部分有大于right部分的，那么逆序数与右侧剩余长度有关；

  1. 归并排序；
  2. 遍历l——>r，判断左侧遍历完时；判断右侧遍历完或者left<=right；判断left>right时；
